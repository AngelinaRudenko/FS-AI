# Flight Price Prediction

Machine learning project for predicting flight prices using Random Forest Regression.

## Project Structure

```
FS-AI/
├── dataset/                      # Flight price datasets
│   ├── business.csv              # Business class flight data
│   ├── economy.csv               # Economy class flight data
│   └── Clean_Dataset.csv         # Cleaned combined dataset
│
├── src/                          # Source code
│   └── flight_price_prediciton.ipynb  # Main analysis notebook
│
├── doc/                          # Documentation and reports
│   ├── Report.pdf                # Final project report
│   ├── Report.tex                # LaTeX source for report
│   └── figures/                  # Generated plots and visualizations
│       └── flight_price_data_profile.html
```
